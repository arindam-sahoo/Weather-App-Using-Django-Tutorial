# Steps to Create the Project

## Step 1
### Create a Virtual Environment
```bash
python -m venv .venv
```
### Activate the Virtual Environment
```bash
.\.venv\Scripts\activate
```
### Make a `.gitignore` file to prevent files getting a Git Commit
```bash
# Virtual Environment
.venv
```

## Step 2
### Install Required Libraries
```bash
pip install django
```
